## GET_FOLLOW_PED_CAM_VIEW_MODE

```c
// 0x8D4D46230B2C353A
int GET_FOLLOW_PED_CAM_VIEW_MODE();
```

Gets the global view mode used by all follow-ped cameras.

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | Third Person Near |
| 1 | Third Person Medium |
| 2 | Third Person Far |
| 3 | Cinematic |
| 4 | First Person |
| 6 | Third Person |
