## IS_MODEL_VALID

```c
// 0xC0296A2EDF545E92
bool IS_MODEL_VALID(Hash modelHash);
```

Returns TRUE if a model with this hash exists and is not a streamed archetype. (The hash of a streamed archetype might only be valid at certain locations on the map.)
