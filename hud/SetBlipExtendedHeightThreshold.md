## SET_BLIP_EXTENDED_HEIGHT_THRESHOLD

```c
// 0xC4278F70131BAA6D
void SET_BLIP_EXTENDED_HEIGHT_THRESHOLD(Blip blip, bool UseExtendedHeightThreshold);
```
