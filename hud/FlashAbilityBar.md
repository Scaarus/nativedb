## FLASH_ABILITY_BAR

```c
// 0x02CFBA0C9E9275CE
void FLASH_ABILITY_BAR(int millisecondsToFlashFor);
```

```
Starts the ability bar to flashing for a set period of time Don't spam it! Or you'll be sorry! PARAMS millisecondsToFlashFor - <= 0 to stop.
```
