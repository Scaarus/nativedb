## SET_MOUSE_CURSOR_STYLE

```c
// 0x8DB8CFFD58B62552
void SET_MOUSE_CURSOR_STYLE(int style);
```

sets the style of the mouse pointer

## style Values:
| Value | Name |
| --- | --- |
| 0 | Arrow Dimmed |
| 1 | Arrow |
| 2 | Hand Open |
| 3 | Hand Grab |
| 4 | Hand Middle Finger |
| 5 | Arrow Left |
| 6 | Arrow Right |
| 7 | Arrow Up |
| 8 | Arrow Down |
| 9 | Arrow Trimming |
| 10 | Arrow Plus |
| 11 | Arrow Minus |
