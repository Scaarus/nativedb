## SET_PLAYER_TARGETING_MODE

```c
// 0xB1906895227793F3
void SET_PLAYER_TARGETING_MODE(int TargetMode);
```

```
set targeting mode (0 = hardlock, 1 = softlock, 2 = freeaim).

Possible values for TargetMode:
| Index | Name |
| --- | --- |
| 0 | Gta Traditional |
| 62 | Assisted Aim |
| 63 | Assisted Freeaim |
| 64 | Freeaim |
```
