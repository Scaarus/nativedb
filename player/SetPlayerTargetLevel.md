## SET_PLAYER_TARGET_LEVEL

```c
// 0x5702B917B99DB1CD
void SET_PLAYER_TARGET_LEVEL(int TargetLevel);
```

set player targeting level (e.g. to increase/lower remote player target threat.

## TargetLevel Values:
| Value | Name |
| --- | --- |
| 0 | Everyone |
| 1 | Strangers |
| 2 | Attackers |
