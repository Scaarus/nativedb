## NETWORK_SET_OVERRIDE_SPECTATOR_MODE

```c
// 0x70DA3BF8DACD3210
void NETWORK_SET_OVERRIDE_SPECTATOR_MODE(bool override);
```

Enables/disables.

By default, players cannot chat with players in the same team when spectating. This allows that to be overridden.
