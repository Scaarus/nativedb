## NETWORK_SESSION_LEAVE

```c
// 0xB9351A07A0D458B1
bool NETWORK_SESSION_LEAVE(int leaveFlags);
```

Leaves a network session (equivalent to [`NETWORK_SESSION_END`](#_0xA02E59562D711006))


## Parameters
* **leaveFlags**: Bit flags built from LEAVE_FLAGS
