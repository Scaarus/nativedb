## FILLOUT_PM_PLAYER_LIST

```c
// 0xCBBD7C4991B64809
bool FILLOUT_PM_PLAYER_LIST(int count, int type);
```

Fills out the list of players in the PM Player List (clears the previous data).

## type Values:
| Value | Name |
| --- | --- |
| 100 | Last Job |
| 250 | Joined |
| 251 | Corona Players |
| 252 | Invitable Session Players |
| 253 | Matched Players |
| 254 | Corona Playlist |
| 255 | Directory |
