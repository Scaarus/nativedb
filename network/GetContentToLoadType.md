## GET_CONTENT_TO_LOAD_TYPE

```c
// 0x8B0C2964BA471961
int GET_CONTENT_TO_LOAD_TYPE();
```

Returns the type of UGC to load.

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | Mission |
| 1 | Mission Playlist |
| 2 | Life Invader Post |
| 3 | Photo |
| 4 | Challenge |
