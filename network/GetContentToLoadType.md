## GET_CONTENT_TO_LOAD_TYPE

```c
// 0x8B0C2964BA471961
ugc_type GET_CONTENT_TO_LOAD_TYPE();
```

```
Returns the type of UGC to load.

Possible return values:
| Index | Name |
| --- | --- |
| 0 | Mission |
| 99 | Mission Playlist |
| 100 | Life Invader Post |
| 101 | Photo |
| 102 | Challenge |
```
