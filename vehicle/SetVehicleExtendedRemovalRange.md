## SET_VEHICLE_EXTENDED_REMOVAL_RANGE

```c
// 0x79DF7E806202CE01
void SET_VEHICLE_EXTENDED_REMOVAL_RANGE(Vehicle vehicle, int removalRange);
```

Sets the extended removal range for the given vehicle, so that it wont be removed inside that range

removalRange is cast to a s16
