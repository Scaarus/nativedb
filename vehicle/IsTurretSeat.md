## IS_TURRET_SEAT

```c
// 0xE33FFA906CE74880
bool IS_TURRET_SEAT(Vehicle vehicle, int seat);
```

Returns if the seat has a turret (one the ped holds) associated with it

## seat Values:
| Value | Name |
| --- | --- |
| -2 | Any Passenger |
| -1 | Driver |
| 0 | Front Right |
| 1 | Back Left (Back Left) |
| 2 | Back Right (Back Right) |
| 3 | Extra Left 1 |
| 4 | Extra Right 1 |
| 5 | Extra Left 2 |
| 6 | Extra Right 2 |
| 7 | Extra Left 3 |
| 8 | Extra Right 3 |
