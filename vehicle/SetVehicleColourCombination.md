## SET_VEHICLE_COLOUR_COMBINATION

```c
// 0x33E8CD3322E2FE31
void SET_VEHICLE_COLOUR_COMBINATION(Vehicle vehicle, int ColourCombination);
```

Set some of the preselcted colour combinations.

Car models have a number of colour combinations preset for them in carvariations.pso.meta Use GET_NUM_CAR_COLOURS first to find out how many combinations there are.
