## GET_VEHICLE_ACCELERATION

```c
// 0x5DD35C8D074E57AE
float GET_VEHICLE_ACCELERATION(Vehicle vehicle);
```

```
Get the maximum drive force this vehicle (with mods) can apply to the ground. This isn't particularly accurate and should be used as a guide and referenced with top speed.
```
