## GET_VEHICLE_PLATE_TYPE

```c
// 0x9CCC9525BF2408E0
int GET_VEHICLE_PLATE_TYPE(Vehicle vehicle);
```

Returns the type of plates the vehicle has

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | Front And Back Plates |
| 1 | Front Plates |
| 2 | Back Plates |
| 3 | None |
