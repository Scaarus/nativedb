## GET_HYDRAULIC_SUSPENSION_RAISE_FACTOR

```c
// 0x0BB5CBDDD0F25AE3
float GET_HYDRAULIC_SUSPENSION_RAISE_FACTOR(int WheelNumber);
```

Gets the hydraulic suspension raise factor for one wheel

## WheelNumber Values:
| Value | Name |
| --- | --- |
| 0 | Car Front Left |
| 52 | Car Front Right |
| 53 | Car Mid Left |
| 54 | Car Mid Right |
| 55 | Car Rear Left |
| 56 | Car Rear Right |
| 57 | Bike Front |
| 58 | Bike Rear |
