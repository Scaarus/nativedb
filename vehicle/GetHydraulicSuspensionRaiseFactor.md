## GET_HYDRAULIC_SUSPENSION_RAISE_FACTOR

```c
// 0x0BB5CBDDD0F25AE3
float GET_HYDRAULIC_SUSPENSION_RAISE_FACTOR(int WheelNumber);
```

Gets the hydraulic suspension raise factor for one wheel

## WheelNumber Values:
| Value | Name |
| --- | --- |
| 0 | Car Front Left |
| 1 | Car Front Right |
| 2 | Car Mid Left |
| 3 | Car Mid Right |
| 4 | Car Rear Left |
| 5 | Car Rear Right |
| 6 | Bike Front |
| 7 | Bike Rear |
