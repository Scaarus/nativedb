## ALLOW_TRAIN_TO_BE_REMOVED_BY_POPULATION

```c
// 0x2310A8F9421EBF43
void ALLOW_TRAIN_TO_BE_REMOVED_BY_POPULATION(Vehicle vehicle);
```

Sets a train vehicle to be removable by population. Only use if the train was created via [`CREATE_VEHICLE`](#_0xAF35D0D2583051B0). If possible use: CREATE_MISSION_TRAIN
