## NETWORK_USE_HIGH_PRECISION_TRAIN_BLENDING

```c
// 0xEC0C1D4922AF9754
void NETWORK_USE_HIGH_PRECISION_TRAIN_BLENDING(Network network, bool UseHighPrecision);
```

Sets whether to use high precision network blending for the specified network ID. High precision blending is useful for small objects that need to be kept very tightly in sync for gameplay purposes


## Parameters
* **network**: 
* **UseHighPrecision**: Flag indicating whether to use high precision or standard network blending
