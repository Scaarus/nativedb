## IS_VEHICLE_DOOR_FULLY_OPEN

```c
// 0x3E933CFF7B111C22
bool IS_VEHICLE_DOOR_FULLY_OPEN(Vehicle vehicle, int DoorNumber);
```

Checks that a vehicle door is open

## DoorNumber Values:
| Value | Name |

| -1 | Invalid |

| 1 | Front Right |

| 3 | Rear Right |
