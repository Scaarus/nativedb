## GET_IS_DOOR_VALID

```c
// 0x645F4B6E8499F632
bool GET_IS_DOOR_VALID(int DoorNumber);
```

returns true if the door is valid

## DoorNumber Values:
| Value | Name |
| --- | --- |
| -1 | Invalid |
| 0 | Front Left |
| 1 | Front Right |
| 2 | Rear Left |
| 3 | Rear Right |
| 4 | Bonnet |
| 5 | Boot |
