## MODIFY_VEHICLE_TOP_SPEED

```c
// 0x93A3996368C94158
void MODIFY_VEHICLE_TOP_SPEED(Vehicle vehicle, float PercentChange);
```

Sets the vehicle's top speed to the specified percent of his previous top speed

PercentChange is a value from -100.0 to 100.0
