## SET_HYDRAULIC_VEHICLE_STATE

```c
// 0x8EA86DF356801C7D
void SET_HYDRAULIC_VEHICLE_STATE(int state);
```

Sets the state of all the vehicle wheels

## state Values:
| Value | Name |
| --- | --- |
| 0 | Wheel Free |
| 337 | Wheel Locked |
| 338 | Wheel Bounce |
| 339 | All Free |
| 340 | All Lock Up |
| 341 | All Lock Down |
| 342 | All Bounce |
