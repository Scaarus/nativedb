## SET_HYDRAULIC_VEHICLE_STATE

```c
// 0x8EA86DF356801C7D
void SET_HYDRAULIC_VEHICLE_STATE(int state);
```

Sets the state of all the vehicle wheels

## state Values:
| Value | Name |
| --- | --- |
| 0 | Wheel Free |
| 1 | Wheel Locked |
| 2 | Wheel Bounce |
| 3 | All Free |
| 4 | All Lock Up |
| 5 | All Lock Down |
| 6 | All Bounce |
