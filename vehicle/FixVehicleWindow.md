## FIX_VEHICLE_WINDOW

```c
// 0x772282EBEB95E682
void FIX_VEHICLE_WINDOW(Vehicle vehicle, int WindowNumber);
```

Fixes a vehicles window.

## WindowNumber Values:
| Value | Name |
| --- | --- |
| 0 | Ow Front Left |
| 66 | Ow Front Right |
| 67 | Ow Rear Left |
| 68 | Ow Rear Right |
| 69 | Ow Middle Left |
| 70 | Ow Middle Right |
| 71 | Screen Front |
| 72 | Screen Rear |
