## GET_VEHICLE_MOD_KIT_TYPE

```c
// 0xFC058F5121E54C32
int GET_VEHICLE_MOD_KIT_TYPE(Vehicle vehicle);
```

Returns the type of the current mod kit onthe specified vehicle. Note: vehicle must have a kit set!

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | MKT_STANDARD |
| 1 | MKT_SPORT |
| 2 | MKT_SUV |
| 3 | MKT_SPECIAL |
