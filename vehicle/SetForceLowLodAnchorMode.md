## SET_FORCE_LOW_LOD_ANCHOR_MODE

```c
// 0xB28B1FE5BFADD7F5
void SET_FORCE_LOW_LOD_ANCHOR_MODE(Vehicle vehicle, bool ForcePlayerLowLodAnchorFlag);
```

Set this to true on boats which should remain in the non-physical, low LOD anchor mode even when a player is driving it.
