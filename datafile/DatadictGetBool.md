## DATADICT_GET_BOOL

```c
// 0x1186940ED72FFEEC
bool DATADICT_GET_BOOL(Any* dict, string name);
```

Gets the value of a boolean dictionary entry


## Parameters
* **dict**: The dictionary object
* **name**: The name of the data item in the dictionary
