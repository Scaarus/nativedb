## ADD_TO_ITEMSET

```c
// 0xE3945201F14637DD
bool ADD_TO_ITEMSET(Item item, Itemset itemset);
```

```
Add an entity or other item to a set.

Each item can only be in the set once, an error will be generated otherwise. Returns TRUE on success.
```
