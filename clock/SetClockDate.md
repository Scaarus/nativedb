## SET_CLOCK_DATE

```c
// 0xB096419DF0D06CE7
void SET_CLOCK_DATE(int d, int m, int y);
```

sets the date to d/m/y

## Values for `m`:
| Value | Name |
| --- | --- |
| 0 | JANUARY |
| 1 | FEBRUARY |
| 2 | MARCH |
| 3 | APRIL |
| 4 | MAY |
| 5 | JUNE |
| 6 | JULY |
| 7 | AUGUST |
| 8 | SEPTEMBER |
| 9 | OCTOBER |
| 10 | NOVEMBER |
| 11 | DECEMBER |


the month is specified in the range 0-11 (0 is January, 11 is December)
