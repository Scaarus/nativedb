## SET_CLOCK_DATE

```c
// 0xB096419DF0D06CE7
void SET_CLOCK_DATE(int d, int m, int y);
```

sets the date to d/m/y

## m Values:
| Value | Name |
| --- | --- |
| 0 | JANUARY |
| 4 | FEBRUARY |
| 5 | MARCH |
| 6 | APRIL |
| 7 | MAY |
| 8 | JUNE |
| 9 | JULY |
| 10 | AUGUST |
| 11 | SEPTEMBER |
| 12 | OCTOBER |
| 13 | NOVEMBER |
| 14 | DECEMBER |


the month is specified in the range 0-11 (0 is January, 11 is December)
