## GET_CLOCK_DAY_OF_WEEK

```c
// 0xD972E4BD7AEB235F
int GET_CLOCK_DAY_OF_WEEK();
```

gets the current day of the week (0 is sunday, 6 is saturday)

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | SUNDAY |
| 1 | MONDAY |
| 2 | TUESDAY |
| 3 | WEDNESDAY |
| 4 | THURSDAY |
| 5 | FRIDAY |
| 6 | SATURDAY |
