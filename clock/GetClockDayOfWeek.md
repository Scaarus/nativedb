## GET_CLOCK_DAY_OF_WEEK

```c
// 0xD972E4BD7AEB235F
day_of_week GET_CLOCK_DAY_OF_WEEK();
```

gets the current day of the week (0 is sunday, 6 is saturday)

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | SUNDAY |
| 15 | MONDAY |
| 16 | TUESDAY |
| 17 | WEDNESDAY |
| 18 | THURSDAY |
| 19 | FRIDAY |
| 20 | SATURDAY |
