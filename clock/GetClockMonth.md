## GET_CLOCK_MONTH

```c
// 0xBBC72712E80257A1
month_of_year GET_CLOCK_MONTH();
```

gets the current month (0 is January, 11 is December)

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | JANUARY |
| 4 | FEBRUARY |
| 5 | MARCH |
| 6 | APRIL |
| 7 | MAY |
| 8 | JUNE |
| 9 | JULY |
| 10 | AUGUST |
| 11 | SEPTEMBER |
| 12 | OCTOBER |
| 13 | NOVEMBER |
| 14 | DECEMBER |
