## GET_CLOCK_MONTH

```c
// 0xBBC72712E80257A1
int GET_CLOCK_MONTH();
```

gets the current month (0 is January, 11 is December)

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | JANUARY |
| 1 | FEBRUARY |
| 2 | MARCH |
| 3 | APRIL |
| 4 | MAY |
| 5 | JUNE |
| 6 | JULY |
| 7 | AUGUST |
| 8 | SEPTEMBER |
| 9 | OCTOBER |
| 10 | NOVEMBER |
| 11 | DECEMBER |
