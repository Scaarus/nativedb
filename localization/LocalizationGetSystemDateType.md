## LOCALIZATION_GET_SYSTEM_DATE_TYPE

```c
// 0xA8AE43AEC1A61314
int LOCALIZATION_GET_SYSTEM_DATE_TYPE();
```

Used to check the language/region date formatting of the system

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | Dmy |
| 1 | Mdy |
| 2 | Ymd |

Returns the current system date formatting type
