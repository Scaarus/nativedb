## GET_PED_PROP_INDEX

```c
// 0x898CC20EA75BACD8
int GET_PED_PROP_INDEX(int Position);
```

Get the id of the prop which has been assigned at the given position for the given ped.

## Position Values:
| Value | Name |
| --- | --- |
| 0 | Head |
| 1 | Eyes |
| 2 | Ears |
| 3 | Mouth |
| 4 | Left Hand |
| 5 | Right Hand |
| 6 | Left Wrist |
| 7 | Right Wrist |
| 8 | Hip |
