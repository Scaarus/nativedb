## GET_PED_PROP_INDEX

```c
// 0x898CC20EA75BACD8
int GET_PED_PROP_INDEX(int Position);
```

```
Get the id of the prop which has been assigned at the given position for the given ped.

Possible values for Position:
| Index | Name |
| --- | --- |
| 0 | Head |
| 131 | Eyes |
| 132 | Ears |
| 133 | Mouth |
| 134 | Left Hand |
| 135 | Right Hand |
| 136 | Left Wrist |
| 137 | Right Wrist |
| 138 | Hip |
```
