## SET_GROUP_FORMATION

```c
// 0xCE2F5FC3AF7E8C1E
void SET_GROUP_FORMATION(Group group, int GroupFormation);
```

Sets the formation a group can take

## GroupFormation Values:
| Value | Name |
| --- | --- |
| 0 | Loose |
| 1 | Surround Facing Inwards |
| 2 | Surround Facing Ahead |
| 3 | Line Abreast |
| 4 | Follow In Line |
