## SET_GROUP_FORMATION

```c
// 0xCE2F5FC3AF7E8C1E
void SET_GROUP_FORMATION(Group group, int GroupFormation);
```

```
Sets the formation a group can take

Possible values for GroupFormation:
| Index | Name |
| --- | --- |
| 106 | Loose |
| 107 | Surround Facing Inwards |
| 108 | Surround Facing Ahead |
| 109 | Line Abreast |
| 110 | Follow In Line |
```
