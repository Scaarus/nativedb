## GET_HASH_OF_MAP_AREA_AT_COORDS

```c
// 0x7EE64D51E8498728
int GET_HASH_OF_MAP_AREA_AT_COORDS(Vector3 Coords);
```

Returns the hash of the name of the area at these coordinates (should be City or Countryside) The returned value should be either MAP_AREA_CITY or MAP_AREA_COUNTRYSIDE
