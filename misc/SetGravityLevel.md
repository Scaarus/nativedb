## SET_GRAVITY_LEVEL

```c
// 0x740E14FAD5842351
void SET_GRAVITY_LEVEL(int iGravityLevel);
```

Changes the current gravity level

## iGravityLevel Values:
| Value | Name |
| --- | --- |
| 0 | Earth |
| 38 | Moon |
| 39 | Low |
| 40 | Zero |


Should be done at the start of a level or after a cutscene. Not advisable to change during normal play.
