## GET_STATUS_OF_MANUAL_SAVE

```c
// 0x397BAA01068BAA96
int GET_STATUS_OF_MANUAL_SAVE();
```

Returns the status of the manual save that is launched by [SET_SAVE_MENU_ACTIVE](#_0xC9BF75D28165FF77)()

## Return Type Values:
| Value | Name |
| --- | --- |
| 0 | Succeeded |
| 1 | In Progress |
| 2 | Failed |
