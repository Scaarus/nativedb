## SET_BIT

```c
// 0x933D6A9EEC1BACD0
void SET_BIT(int Variable, int BitIndex);
```

Sets the bit at the index (BitToSetIndex) of the integer (IntToModify) passed.

The index (BitToSetIndex) passed must be between 0 and 31 or the command will ASSERT.
