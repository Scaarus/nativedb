## SET_ENTITY_LIGHTS

```c
// 0x7CFBA6A80BDF3874
void SET_ENTITY_LIGHTS(Entity entity, bool LightSwitch);
```

Sets on all lights attached to the entity; bLightSwitch works in "inversed" logic manner: FALSE-to ENABLE entity lights, TRUE-to DISABLE entity lights;
