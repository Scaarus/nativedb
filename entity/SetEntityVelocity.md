## SET_ENTITY_VELOCITY

```c
// 0x1C99BB7B6E96D16F
void SET_ENTITY_VELOCITY(Entity entity, Vector3 VelocityComponentt);
```

```
Sets the entity's current velocity.

The velocity componets should be less than 500.

The speeds are measured in metres per second.
```
