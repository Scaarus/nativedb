## ENABLE_ALL_CONTROL_ACTIONS

```c
// 0xA5FFE9B05F199DE7
void ENABLE_ALL_CONTROL_ACTIONS(int control);
```

Enables all the inputs for a player control.

## Values for `control`:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
