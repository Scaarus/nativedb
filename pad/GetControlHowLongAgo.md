## GET_CONTROL_HOW_LONG_AGO

```c
// 0xD7D22F5592AED8BA
int GET_CONTROL_HOW_LONG_AGO(int control);
```

Return how long ago a control last had a value (in milliseconds).

## control Values:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
