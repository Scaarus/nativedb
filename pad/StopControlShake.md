## STOP_CONTROL_SHAKE

```c
// 0x38C16A305E8CDC8D
void STOP_CONTROL_SHAKE(int control);
```

Stops the given Pad, assigned to a Control, from shaking.

## control Values:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
