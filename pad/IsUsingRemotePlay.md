## IS_USING_REMOTE_PLAY

```c
// 0x23F09EADC01449D6
bool IS_USING_REMOTE_PLAY(int control);
```

Returns true if the the player is playing on a remote play device.

## control Values:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
