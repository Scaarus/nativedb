## CLEAR_CONTROL_SHAKE_SUPPRESSED_ID

```c
// 0xA0CEFCEA390AAB9B
void CLEAR_CONTROL_SHAKE_SUPPRESSED_ID(int control);
```

Clears a supress id passed to [`SET_CONTROL_SHAKE_SUPPRESSED_ID`](#_0xF239400E16C23E08).

## control Values:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
