## DISABLE_ALL_CONTROL_ACTIONS

```c
// 0x5F4B6931816E599B
void DISABLE_ALL_CONTROL_ACTIONS(int control);
```

Disables all the inputs for a player control.

## control Values:
| Value | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
