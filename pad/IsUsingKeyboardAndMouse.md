## IS_USING_KEYBOARD_AND_MOUSE

```c
// 0xA571D46727E2B718
bool IS_USING_KEYBOARD_AND_MOUSE(int control);
```

```
Returns true if the player is using keyboard and mouse controls.

Possible values for control:
| Index | Name |
| --- | --- |
| 0 | PLAYER_CONTROL |
| 1 | CAMERA_CONTROL |
| 2 | FRONTEND_CONTROL |
```
