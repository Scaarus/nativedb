## SET_CURSOR_POSITION

```c
// 0xFC695459D4D0E219
bool SET_CURSOR_POSITION(float x, float y);
```

Sets the mouse cursor position on the PC version. NOTES: Returns true if the cursor position was succesfully set (also returns false on platforms not supported).
